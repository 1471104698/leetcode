## **[1206. 设计跳表](https://leetcode-cn.com/problems/design-skiplist/)**

```java
不使用任何库函数，设计一个跳表。

跳表是在 O(log(n)) 时间内完成增加、删除、搜索操作的数据结构。跳表相比于树堆与红黑树，其功能与性能相当，并且跳表的代码长度相较下更短，其设计思想与链表相似。

例如，一个跳表包含 [30, 40, 50, 60, 70, 90]，然后增加 80、45 到跳表中，以下图的方式操作：
```

![img](https://assets.leetcode.com/uploads/2019/09/27/1506_skiplist.gif)

```java
跳表中有很多层，每一层是一个短的链表。在第一层的作用下，增加、删除和搜索操作的时间复杂度不超过 O(n)。跳表的每一个操作的平均时间复杂度是 O(log(n))，空间复杂度是 O(n)。

在本题中，你的设计应该要包含这些函数：

bool search(int target) : 返回target是否存在于跳表中。
void add(int num): 插入一个元素到跳表。
bool erase(int num): 在跳表中删除一个值，如果 num 不存在，直接返回false. 如果存在多个 num ，删除其中任意一个即可。
了解更多 : https://en.wikipedia.org/wiki/Skip_list

注意，跳表中可能存在多个相同的值，你的代码需要处理这种情况。
```





## **样例:**

```java
Skiplist skiplist = new Skiplist();

skiplist.add(1);
skiplist.add(2);
skiplist.add(3);
skiplist.search(0);   // 返回 false
skiplist.add(4);
skiplist.search(1);   // 返回 true
skiplist.erase(0);    // 返回 false，0 不在跳表中
skiplist.erase(1);    // 返回 true
skiplist.search(1);   // 返回 false，1 已被擦除

约束条件:
0 <= num, target <= 20000
最多调用 50000 次 search, add, 以及 erase操作。
```





## 实现思路

```java

    https://juejin.im/post/57fa935b0e3dd90057c50fbc

/*跳表作为一种数据结构通常用于取代平衡树（现在都是使用红黑树，AVL 树基本没怎么用）*/。
平衡树可以用于表示抽象的数据类型如字典和有序链表，
它通过树旋转（Tree Rotation）操作强制使树结构保持平衡来保证节点搜索的效率。在数据为随机插入的情况下，平衡树性能表现良好；
但数据为顺序插入或者需要删除节点的情况下，平衡树的性能就会有些糟糕。

跳表可以作为平衡树的一种替代选择。它使用随机的平衡策略取代平衡树严格的强制的树平衡策略。因此它具有更简单有效的插入/删除方法以及更快的搜索速度。

跳表的 查询、删除、添加都是 O(logn)
```

## 实现代码

```java
class Skiplist {
    //跳表节点
    class SkipNode{
        //当前节点层数，最底层为 1 层
        int level;
        //节点值
        int val;
        //指向下一个节点
        SkipNode next;
        //指向下一层同一个节点
        SkipNode lower;
        public SkipNode(int val, int level){
            this.val = val;
            this.level = level;
        }
    }
    //最高层次
    int maxLevel = 16;
    //最上层跳表头
    SkipNode head;
    public Skiplist() {
        head = new SkipNode(-1, 1);
    }
    
    /*
    如果当前节点为目标节点，那么直接返回 true
    当下一个节点为 空 或者 大于 target 时，移动到下一层
    当下一个节点小于等于 target 时，移动到下一个节点
    当到达底层节点 并且 下一个节点为空 或者 下一个节点大于 target 时，会继续进入下一层，这时为 null，返回 false
    */
    public boolean search(int target) {
        SkipNode cur = head;
        while(cur != null){
            if(cur.val == target){
                return true;
            }
            if(cur.next == null || cur.next.val > target){
                cur = cur.lower;
            }else{
                cur = cur.next;
            }
        }
        return false;
    }

    Random random = new Random();

    public void add(int num) {
        //随机获取当前节点的层数
        int level = randLevel(0.5);

        //已有层数 比 目标层数小，那么需要向上扩充
        while(head.level < level){
            //创建新节点
            SkipNode newHead = new SkipNode(-1, head.level + 1);
            newHead.lower = head;
            head = newHead;
        }

        SkipNode cur = head;
        //找到 level 对应的层
        while(cur.level > level){
            cur = cur.lower;
        }
        //上一层的 目标 num 节点，用于跟下一层进行建立连接
        SkipNode pre = null;

        while(cur != null){
            //从当前层找一个适合 num 插入的位置
            while(cur.next != null && cur.next.val < num){
                cur = cur.next;
            }
            SkipNode node = new SkipNode(num, level);
            //插入
            node.next = cur.next;
            cur.next = node;
            //进入下一层
            cur = cur.lower;
            //进行当前层和上一层的连接
            if(pre == null){
                pre = node;
            }else{
                pre.lower = node;
                pre = node;
            }
        }
        
    }


    public int randLevel(double p) {
        int level = 1;

        while (Math.random() < p && level < maxLevel){
            level++;
        }

        return level;
    }
    
    public boolean erase(int num) {
        //节点查找
        SkipNode cur = head;

        while(cur != null){
            //如果当前节点的下一个节点值为 num，那么退出循环
            if(cur.next != null && cur.next.val == num){
                break;
            }
            if(cur.next == null || cur.next.val > num){
                cur = cur.lower;
            }else{
                cur = cur.next;
            }
        }
        //如果 cur == null，表示不存在 num 这个节点
        if(cur == null){
            return false;
        }

        while(cur != null){
            //只要当前存在 num 这个节点，那么下面的层都必定存在，即不会遇到 cur.next == null 的情况，而是必定会先遇到 num
            while(cur.next.val != num){
                cur = cur.next;
            }
            //删除当前层的 num 节点
            cur.next = cur.next.next;
            //移动到下一层
            cur = cur.lower;
        }
        return true;
    }
}
```
