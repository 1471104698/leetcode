## **[970. 强整数](https://leetcode-cn.com/problems/powerful-integers/)**

```java
给定两个正整数 x 和 y，如果某一整数等于 x^i + y^j，其中整数 i >= 0 且 j >= 0，那么我们认为该整数是一个强整数。

返回值小于或等于 bound 的所有强整数组成的列表。

你可以按任何顺序返回答案。在你的回答中，每个值最多出现一次。

提示：

1 <= x <= 100
1 <= y <= 100
0 <= bound <= 10^6
```



## **示例 1：**

```java
输入：x = 2, y = 3, bound = 10
输出：[2,3,4,5,7,9,10]
解释： 
2 = 2^0 + 3^0
3 = 2^1 + 3^0
4 = 2^0 + 3^1
5 = 2^1 + 3^1
7 = 2^2 + 3^1
9 = 2^3 + 3^0
10 = 2^0 + 3^2
```



## **示例 2：**

```java
输入：x = 3, y = 5, bound = 15
输出：[2,4,6,8,10,14]
```





## **实现思路**

```java
返回所有小于等于 bound 的强整数

我们使用一个 boolean 数组 exist 存储 [2, bount] 中已经访问过的数，防止重复添加

//循环：
1、外层循环，我们从 x^0 开始，先判断 x^0 对应的 y 的情况，循环结束条件： xsum <= bound - 1，因为最少必须加上 y^0 = 1
2、内层循环，我们从 y^0 开始，计算出能和 xsum 相加不超过 bound 的数，循环结束条件: xsum + ysum <= bound

//注意：
1、1 的 n 次方都是 1，因此如果 x == 1 或 y == 1，那么我们只需要循环一次即可
```



## **实现代码**

```java
class Solution {
    public List<Integer> powerfulIntegers(int x, int y, int bound) {
        List<Integer> res = new ArrayList<>();

        //记录某个值是否存储过
        boolean[] exist = new boolean[bound + 1];

        for(int xsum = 1; xsum <= bound - 1; xsum *= x){
            for(int ysum = 1; xsum + ysum <= bound; ysum *= y){
                int sum = xsum + ysum;
                if(!exist[sum]){
                    exist[sum] = true;
                    res.add(sum);
                }
                //1 的 n 次方都是 1，因此如果 y == 1，那么只需要计算一个结果即可
                if(y == 1){
                    break;
                }
            }
            //1 的 n 次方都是 1，因此如果 y == 1，那么只需要计算一个结果即可
            if(x == 1){
                break;
            }
        }
        return res;
    }
}
```
