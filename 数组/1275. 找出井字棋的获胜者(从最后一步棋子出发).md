# [1275. 找出井字棋的获胜者(从最后一步棋子出发)](https://leetcode-cn.com/problems/find-winner-on-a-tic-tac-toe-game/)

```java
A 和 B 在一个 3 x 3 的网格上玩井字棋。

井字棋游戏的规则如下：

玩家轮流将棋子放在空方格 (" ") 上。
第一个玩家 A 总是用 "X" 作为棋子，而第二个玩家 B 总是用 "O" 作为棋子。
"X" 和 "O" 只能放在空方格中，而不能放在已经被占用的方格上。
只要有 3 个相同的（非空）棋子排成一条直线（行、列、对角线）时，游戏结束。
如果所有方块都放满棋子（不为空），游戏也会结束。
游戏结束后，棋子无法再进行任何移动。
给你一个数组 moves，其中每个元素是大小为 2 的另一个数组（元素分别对应网格的行和列），它按照 A 和 B 的行动顺序（先 A 后 B）记录了两人各自的棋子位置。

如果游戏存在获胜者（A 或 B），就返回该游戏的获胜者；如果游戏以平局结束，则返回 "Draw"；如果仍会有行动（游戏未结束），则返回 "Pending"。

你可以假设 moves 都 有效（遵循井字棋规则），网格最初是空的，A 将先行动。
```



## **示例 1：**

```java
输入：moves = [[0,0],[2,0],[1,1],[2,1],[2,2]]
输出："A"
解释："A" 获胜，他总是先走。
"X  "    "X  "    "X  "    "X  "    "X  "
"   " -> "   " -> " X " -> " X " -> " X "
"   "    "O  "    "O  "    "OO "    "OOX"
```



## **示例 2：**

```java
输入：moves = [[0,0],[1,1],[0,1],[0,2],[1,0],[2,0]]
输出："B"
解释："B" 获胜。
"X  "    "X  "    "XX "    "XXO"    "XXO"    "XXO"
"   " -> " O " -> " O " -> " O " -> "XO " -> "XO " 
"   "    "   "    "   "    "   "    "   "    "O  "
```



## **示例 3：**

```java
输入：moves = [[0,0],[1,1],[2,0],[1,0],[1,2],[2,1],[0,1],[0,2],[2,2]]
输出："Draw"
输出：由于没有办法再行动，游戏以平局结束。
"XXO"
"OOX"
"XOX"
```





## 方法①、只判断下最后一步的棋手

### 实现思路

```java
这道题是按照规则来下棋的，即如果还没有人赢，那么会一直下到某人赢 或者 平局为止
而题目给出的有 3 种情况：最后一步某人赢了，平局了，棋局还没有结束

我们只需要记录最后一个人的下棋情况是否能够赢，因为如果前面一个人那一步赢了，不会存在这一步棋
而要进行判断，我们可以进行倒序遍历，每次 i -= 2，只看这个人的下棋情况
同时记录 每行每列以及两条对角线 的 这个人的 棋的个数

关于对角线的判断，当 i - j == 0 斜向下主对角线，i + j == len - 1 是斜向上主对角线
```



### 实现代码

```java
class Solution {
    public String tictactoe(int[][] moves) {
       
        int len = moves.length;
        
        int[] col = new int[3];
        int[] row = new int[3];
        //两条主对角线
        int[] diag = new int[2];
		
        //胜利时连续的棋数
        int winNum = 3;
        
        for(int i = len - 1; i >= 0; i -= 2){
            int[] p = moves[i];
            row[p[0]]++;
            col[p[1]]++;

            if(p[0] - p[1] == 0){
                diag[0]++;
            }
            if(p[0] + p[1] == 2){
                diag[1]++;
            }
			
            //当某条线上的存在连续 winNum 个棋子时，玩家胜利
            if(row[p[0]] == winNum || col[p[1]] == winNum || diag[0] == winNum || diag[1] == winNum){
                //根据步数来判断谁是胜利者
                return len % 2 != 0 ? "A" : "B";
            }
        }
		
        //如果棋盘满了，表示是平局，否则棋局还没结束
        return len == 9 ? "Draw" : "Pending";
    }
}
```
