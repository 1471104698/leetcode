## **[1081. 不同字符的最小子序列](https://leetcode-cn.com/problems/smallest-subsequence-of-distinct-characters/)**

```java
返回字符串 text 中按字典序排列最小的子序列，该子序列包含 text 中所有不同字符一次。
```



## **示例 1：**

```java
输入："cdadabcc"
输出："adbc"
```



## **示例 2：**

```java
输入："abcd"
输出："abcd"
```



## **示例 3：**

```java
输入："ecbacba"
输出："eacb"
```
